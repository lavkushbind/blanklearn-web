# **App Name**: EduConnect

## Core Features:

- Gisty Meet Integration: Seamless Gisty Meet integration for live video conferencing.
- Student Roster Management: Manage student enrollment, limited to 10 students per class.
- Screen Sharing: Enable teachers to share their screen for presentations and demonstrations.
- "Raise Hand" Management: System to signal when a student wants to speak.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability, referencing the focused atmosphere of learning environments.
- Background color: Light gray (#F0F2F5), a low saturation version of the primary hue, provides a neutral backdrop that minimizes distraction and allows educational content to stand out.
- Accent color: Warm yellow (#FFC107) for interactive elements. This hue, approximately 30 degrees from blue on the color wheel, adds an inviting and approachable feel to calls-to-action, which can complement the main colors of a dark color scheme.
- Clean and readable sans-serif font optimized for screen use.
- Consistent, modern icons for key features and navigation.
- Intuitive and straightforward layout for easy navigation and user experience.
